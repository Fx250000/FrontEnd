Common CSS Properties

Property Example Description
color color: blue; Sets the color of the text.
background-color background-color: yellow; Sets the background color of an element.
font-size font-size: 16px; Sets the size of the text.
margin margin: 10px; Sets the outer spacing of an element.
padding padding: 15px; Sets the inner spacing of an element.
border border: 1px solid black; Sets the border around an element.
text-align text-align: center; Aligns the text horizontally within an element.
